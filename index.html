CodeMirror PCRE mode

This is a CodeMirror mode that brings syntax highlighting for Perl Compatible Regular Expressions (PCRE).

MIME types defined:

  • text/x-regex
  • text/x-pcre-regex

Options:

  • extended: boolean: initial state of the 'x' flag; default: true.
